An auxiliary random variable is been chosen ideally so that it is easy for us to simulate random numbers. For example, you can use the method of inverse transforms because there is a computable inverse.

(5) The Definition of Realization

The realization of a random variable is an observed value, which is the value we actually observed form that random variable.
