Python程序设计2 变量和数学表达式

我们在上一章节已经通过Karel接触了Python和它的基本语句，比如for，while和if等等，我们还介绍了如何去写函数。下面我们需要更进一步的学习Python。

一、打印

为了将我们程序的结果输出在计算机的屏幕上，我们需要一个函数，这个函数的名字是print()。下面来试试我们的第一个程序，它可以将hello world打印在屏幕上：

def main():
    print("hello, world!")

# This provided line is required at the end of a Python
# file to call the main() function.

if __name__ == '__main__':
    main()

运行该程序，我们可以得到输出：

hello, world!

二、input函数

下面的一段程序可以让我们对两个由用户输入的数字做加法运算。由于其中需要用户输入这两个数字，所以我们需要一个能够让用户进行输入的函数，它就是input函数：

def main():
    print("This program adds two numbers.")
    num1 = input("Enter first number: ")
    num1 = int(num1)
    num2 = input("Enter second number: ")
    num2 = int(num2)
    total = num1 + num2
    print("The total is " + str(total) + ".")

# This provided line is required at the end of a Python
# file to call the main() function.

if __name__ == '__main__':
    main()

如果我们想要输出一串字符，那么我们使用 "xxxx"还是 'xxxx'在python的语法里面都是可行的，例如：

print("hello, world")

等同于：

print('hello, world')

不过，当我们想要输出双引号作为字符，那么我们需要在外侧使用单引号，在内侧使用双引号，例如：

print('hello, world: "python"')

上面的语句会输出：

hello, world: "python"

三、变量

在计算机科学中，一个用来存储信息的单位被称为变量。在python中，我们用符号 = 来赋值一个变量，例如：

x = 10

或者：

x = 5

或者：

x = 5 + 7

我们也可以用一个变量来赋值它自己，比如，我们在之前的程序中写过的语句：

count = 0
count = count + 1

对于变量的名称，python语言中也有一些特殊规定：

  • 由字母开始或者由_（下划线）开始
  • 只包括字母、数字、下划线
  • 不能够是python中的保留字（build-in command）
  • 大小写敏感：hello和Hello两个变量是不同的
  • 一个优秀的变量名称应该和它存储的值的含义相关

四、对象和变量类型

当我们创建完一个变量之后，它就变成了一个Python的对象。一个对象可以是大小或者类型不同的任何变量。对象可以被简单理解成电脑内存上的包裹，包裹里面存储着信息，包裹外面贴着一个标签，上面的内容就是我们的变量名称。

当我们在一个包裹里面放入某个变量的时候，我们并不需要特别告诉包裹这个变量的类型是int，float还是其他类型。包裹可以自动地识别变量类型，例如：

  • int类型（无小数点）
x = 10
y = -2
  • float类型（有小数点）
x = 5.0
y = -3.7
  • string类型（单引号/双引号之间有字符/字符串）
x = "hello"
y = 'hello'
  • bool类型（True/False）
x = True
y = False

四、算数运算符

python常见的算数运算符有： +（加法）、 -（减法）、 *（乘法）、 /（除法）、 //（除法取整）、 %（取模）、 **（指数运算）、 -（取反）。关于这些运算符的使用，在这里不再赘述。

在Python中，不同类型的数值是可以相互运算的，但是结果的变量类型也会随着运算的结果改变，比如：

  • 两个整数相加一定是整数（num3）：
num1 = 2
num2 = 3
num3 = num1 + num2
  • 一个整数和一个浮点数相加一定是浮点数（num3）：
num1 = 1
num2 = 1.7
num3 = num1 + num2
  • 指数运算的结果类型取决于具体的数值：

（1）第一种情况：结果（num3）为整数

num1 = 2
num2 = 1
num3 = num1 ** num2

（2）第二种情况：结果（num3）为浮点数

num1 = 2
num2 = -1
num3 = num1 ** num2

要注意到的是，浮点数并不一定总是精确的，例如在计算机中：

1.0 == 0.999999...

在做自加自减运算的时候，我们有方法可以简化我们输入的内容，例如：

num = num + 1

等同于：

num += 1

同样地，

num = num - 1

等同于：

num -= 1

另外，

num = num * 1

等同于：

num *= 1

以及，

num = num / 1

等同于，

num /= 1

五、定义常数

在程序函数外定义全部函数中使用的常数可以让程序更容易阅读，这个常数的名称通常为全部大写。函数的定义通常在函数定义之上。比如：

PI = 3.14

或者：

INCHES_IN_FOOT = 12

六、Math包

python中还包括了一个自带的math包，在程序一开始import这个包就可以进行更多的数学运算操作。我们通过import命令在程序的最开始引用这个包文件：

import math

math包文件中有一些常见的函数，其中最常用的是：

  • 圆周率pi
math.pi
  • 数学常数e
math.e

此外，math包还定义了一些新的运算，例如：

  • 开方
math.sqrt()
  • 以e为底数的指数运算
math.exp()
  • ln运算
math.log()

七、随机数

在计算机科学中，并没有真正的随机数，所谓的随机数都是通过某些算法特定生成的类似于随机数的数字，他们被称为伪随机数（pseudorandom numbers）。

随机数的生成依赖于一个被称为seed的变量，当seed相同的时候，给定的某个随机数生成算法会给出固定的数值。通常情况下，许多生成器的seed是时间，不过也可能是其他任何东西。同样，对于我们来说，我们也可以给定seed让随机数生成器生成固定的数值。

random是python自带的package，我们可以直接在程序之前用import导入它。

import random

除此以外，这个使用这个包的方法如下：

  • 在给定范围min～max之间生成随机整数
random.randint(min, max)
  • 生成0～1之间的随机数
random.random()
  • 在给定范围min～max之间生成随机浮点数
random.uniform(min, max)
  • 设置random的seed
random.seed(x)

"""
File: compute_interest.py
-------------------------
Add your comments here.
"""


def main():

    balance = float(input("Initial balance: "))
    s_yr = int(input("Start year: "))
    s_mo = int(input("Start month: "))
    e_yr = int(input("End year: "))
    e_mo = int(input("End month: "))

    if s_yr > e_yr:
        print("Starting date needs to be before the ending date.")
        return
    elif s_yr == e_yr:
        if s_mo > e_mo:
            print("Starting date needs to be before the ending date.")
            return

    while True:
        r = float(input("Interest rate (0 to quit): "))
        if r == 0:
            break
        else:

            i = s_yr
            j = s_mo
            c_bal = balance

            while i <= e_yr:
                if i < e_yr:
                    for t in range(12):
                        if j <= 12:
                            print("Year", i, ", month", j, " balance:", c_bal)

                            j += 1
                            c_bal *= (1 + r)

                        else:
                            i += 1
                            j = 1
                            break
                else:
                    for t in range(e_mo):
                        print("Year", i, ", month", j, " balance:", c_bal)

                        j += 1
                        c_bal *= (1 + r)

                        if j > e_mo:
                            break

                    break
