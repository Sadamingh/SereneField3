Wacom Drawing Tablet 产品参数

对于Drawing Tablet，除了Wacom之外其他品牌也有很多，比如Huion（绘王中国）UGEE（友基中国），HANWANG（汉王中国）、ViewSonic（优派中国）。但是这些品牌的产品不会在这里讨论。未出现的型号为2020年之前的产品，我们在这里不去讨论。

下面的顺序是按照产品的专业性进行排序的，越靠近末尾的产品意味着这个产品具有更高的专业性和当然，相对来说更高的价格。

Bamboo Slate系列

纸张绘画产品，绘画者需要在纸制品上进行绘画。该产品可以将纸张上的绘画自动同步到其他设备。笔和板是必须的，但是纸张没有任何要求限制。类似于Scanner的功能，不过随着最近手机Scanner软件越来越多，这一产品的实际意义就非常迷惑了。

Bamboo Folio系列

产品功能和Slate系列完全相同，但是多了一个保护盖用来保护纸张。

One by Wacom系列

入门级产品，型号为CTL-472，可用区域为6.0 x 3.7英寸。无线无电池笔压感为2048。解析度为2540。采用USB接口连接，机身接口为Micro-USB。不可触控。

Wacom Intuos系列

入门级产品。分为S号（6.0 x 3.7英寸，蓝牙可选）和M号（8.5 x 5.3英寸，带蓝牙）。不带蓝牙S号仅有黑色，型号为CTL-4100。带蓝牙S号（CTL-4100WL）和M号（CTL-6100WL）都有开心果绿色和黑色两种。无线无电池笔为Wacom Pen 4K，压感为4096。解析度为2540。机身接口为Micro-USB。四个快捷键。不可触控。

Wacom Intuos Pro系列

中级产品。无纸板分为S号（PTH-460，6.3 x 3.9 in）、M号（PTH-660，8.7 x 5.8 in）、以及L号（PTH-860，12.1 x 8.4 in）。有纸版本添加了类似于Slate的功能，分为M号（PTH-660P，8.7 x 5.8 in）、以及L号（PTH-860P，12.1 x 8.4 in）。均带蓝牙。无线无电池笔为Wacom Pro Pen 2，压感为8192，带倾斜60度压感。机身接口为Type-C USB。解析度为5080。纸张模式需要Wacom Finetip。S号为6个快捷键，M和L为8个快捷键。不可触控。

Wacom One系列

中级产品。屏幕为13.3英寸，分辨率1920 x 1080 像素，型号为DTC-133。有线连接，接口为USBA接口和Hdmi接口。无线无电池笔为Wacom One Pen，压感为4096，带倾斜60度压感。机身接口为Micro-USB。无快捷键，不可触控。

Wacom Cintiq系列

中高级产品。屏幕分为16英寸（实际13.6 x 7.6英寸，型号为DTK-1660）以及22英寸（实际18.7 x 10.5英寸，型号为DTK-2260）。分辨率1920 x 1080 像素。无线无电池笔为Wacom Pro Pen 2，压感为8192，带倾斜60度压感。有线连接，接口为USBA接口和Hdmi接口。两个侧边开关，不可触控。

Wacom Cintiq Pro系列

高级产品。共有四种尺寸，分别为13英寸（实际11.6 x 6.5英寸）、16英寸（实际13.6 x 7.6 英寸）、24英寸（实际20.55 X 11.57 英寸）、32英寸（实际27.44 X 15.43 英寸）。除了13寸的分辨率为HD 1920 x 1080像素之外，其他屏幕分辨率都为UHD 4K 3840 x 2160。有线连接，接口为USB-A接口、Hdmi接口、以及USB-typeC接口。机身接口为3个Type C USB。无线无电池笔为Wacom Pro Pen 2，压感为8192，带倾斜60度压感。两个侧边开关加橡皮擦按钮。支持多点触控。

Wacom MobileStudio Pro系列

本身是一个电脑。共有两种尺寸，分别是13英寸（型号为DTHW1321H，实际11.6 x 6.5英寸）和16英寸（型号为DTHW1621H，实际13.6 x 7.6英寸）。13寸屏幕分辨率为WQHD 2560 x 1440像素，16寸屏幕分辨率为UHD 4K 3840 x 2160。处理器为英特尔酷睿i7–8559U，13英寸显卡为英特尔®Iris™Plus Graphics 655，16英寸显卡为NVIDIA® Quadro® P1000（16英寸另外包括一个4GB GDDR5的显存）。硬盘空间为PCIe Gen3 512GB，系统为Windows 10专业版。支持Wifi和蓝牙5.0。接口数量和类型类似于标准笔记本电脑。支持多点触控。
