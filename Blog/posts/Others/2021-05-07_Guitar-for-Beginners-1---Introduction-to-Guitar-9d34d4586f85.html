  1. Parts of the Guitar
  • Body: the main part of the guitar
  • Soundboard: the top piece of wood on the body
  • Soundhole: the round part on the middle of the body
  • Rosette: the decorated part along the edge of the soundhole.
  • Bridge: the wood used to connect the strings to the soundboard
  • Saddle: means a small piece of hard material like a piece of bone that supports the strings on the bridge
  • Strings: means the vibrating elements that produce sound
  • Neck: the board where we put our left-hand fingers
  • Fretboard: on top of the neck, traditionally made by rosewood or maple
  • Headstock: the top part of the guitar where we can modify the tuning pegs
  • Tuning Pegs: the parts by which we can change the tunes of the strings
  • Nut: a nut is a small piece of hard material like a piece of bone that supports the strings at the end closest to the headstock or scroll
  • Metal Frets: they are the things that allow us to perform different pitches

2. Name of the Strings

A classical guitar has six nylon strings. The three low strings are nickel round-wound, but inside they are actually nylon strings. Each of the strings has a letter name and a number name.

  • First-string: High E string as string number 1 is also called the E string
  • Second-string: B string as string number 2
  • Third-string: G string as string number 3
  • Fourth-string: D string as string number 4
  • Fifth-string: A string as string number 5
  • Sixth-string: Another E string as string number 6

3. Finger placement Principles

(1) Right hand

Use the stings as a guild about where fingers would like to go.

  • Thumb on the E string (#6)
  • First finger on the G string (#3)
  • Second finger on the B string (#2)
  • The third finger on the E string (#1)

This is just like the home base we have for typing the keyboard keys on the guitar.

(2) Left hand

The general rule is that we have to use one finger per fret.

  • First finger on the first fret
  • Second finger on the second fret
  • Third finger on the third fret
  • Fourth finger on the fourth fret

5. PIMA Picking Pattern

Right-hand finger patterns like this one should be a part of your daily practice routine. Spend a couple of minutes a day on this exercise for the next few weeks to get ready for more advanced patterns.

The right-hand fingers are symbolized by letters:

p = thumb; 
i = index; 
m = middle; 
a = ring;

PIMA comes from the names of the fingers in Spanish:

p = pulgar; 
i = indicio; 
m = medio; 
a = anular;

6. Picado Technique

Picado is the Spanish guitar technique of playing the melody or single-note lines. It is also known as the rest stroke. It is called so because after your finger plucks the string, it comes to rest on the string behind it. In this way, the finger pushes the string slightly downward instead of upward. The effect, if done correctly creates a stronger louder note. Use just your index and middle fingers (right hand). Remember to: 1. plant, 2. push, 3. alternate, 4. rest.

Finally, let’s try Picado of natural notes on the first string. There are called natural notes because they don’t have sharps or flats.
