Linux操作系统实践2 Unix-like的基本命令行指南

一、列出文件

我们先打开terminal，在终端输入：

ls

这一指令会返回当前目录下的所有文件名称。

如果我们想要查询每个文件的详细信息，我们在ls后面加上-l即可：

ls -l

这一指令会返回当前目录下的文件的所有信息。

我们可能不想要列出所有文件的信息，比如，我们可能只需要列出dataset.xlsx 这个文件的信息。那么我们就可以输入下面的语句，在输出的结果中仅仅显示dataset.xlsx的文件信息：

ls -l dataset.xlsx

有的时候我们可能要选择多个文件，这些文件可能具有比较类似的名字。例如，我们要显示dataset.xlsx, dataset1.xlsx, dataset2.xlsx, 和dataset3.xlsx。但是在后面打出他们的全称太麻烦了，我们就可以利用符号*指代后面的任何内容，比如：

ls -l data*

二、更改目录

我们先打开terminal，通过 ls -l 找到权限为d开头的文件。比如在我们的例子中，files和images都是权限为d开头的文件。

我们可以通过输入下面的指令进入这个文件：

cd files

我们就可以通过这个命令来进入目录。

一些常见的目录符号：～表示home目录，/表示root目录。home目录表示的实际上是我们在何处打开了这个shell。

为了显示全部路径，我们可以输入下面的指令查看当前目录的具体路径：

pwd

通过返回的值，我们可以得到当店目录的路径（Path）。

下面两种命令可以让我们随时返回home目录：

cd ~
cd

下面一种命令可以让我们随时返回根目录：

cd /

下面的命令可以让我们返回上一层目录。比如，如果我们在files目录中，该命令会让我们返回new目录：

cd ..

三、创建目录

我们现在回到了new目录下，我们想要在new目录下创建一个新的目录，我们可以通过mkdir（make directory）指令来实现这一功能：

mkdir mydir

之后我们通过ls从查看当前目录，可以发现new中新增了mydir目录。

我们可以用Tab键对其进行自动填充，例如我们想要进入mydir这个文件夹，我们只需要输入：

cd m

之后我们可以点击Tab键来自动填充后面的内容。

四、移动和复制文件

经过第三部分的操作，我们将dataset.xlsx, dataset1.xlsx, dataset2.xlsx, 和dataset3.xlsx四个文件移动到新建的目录mydir中。我们先回到new目录，然后输入下面的语句：

mv data* mydir

之后我们可以通过在new目录下的ls命令和在mydir目录下使用ls命令来检查。检查的结果表示，四个data文件已经被移动到mydir目录下了。

如果我们想要将这些文件移动回上一层目录，那么只需要在mydir目录下输入命令，其中「..」表示上一层目录：

mv data* ..

如果我们在new目录下，我们想要做同样的事情，我们需要输入下面的命令，其中「.」表示的是当前目录。

mv mydir/data* .

现在我们想要在目录mydir中复制一个dataset，并且将它的文件名命名为dataset4.xlsx，那么我们需要做的是输入下面的命令（形式为：cp + 源文件 + 目标目录）：

cp dataset.xlsx mydir/

之后我们可以发现，在mydir目录下创建了一个dataset.xlsx文件。

如果我们想在new目录下复制一个dataset.xlsx，并将这个新文件文件名作为dataset-new.xlsx那么我们可以输入下面的指令（形式为：cp + 源文件 + 新文件名）。

cp dataset.xlsx dataset-new.xlsx

但是有的时候，我们不仅想要指明复制的目标位置，也想要在复制的同时更改文件的名称。例如，我们想要复制dataset.xlsx到mydir中，并将其复制后的文件名更改为dataset-test.xlsx。那么我们就可以在目录new中通过下面的命令（形式为：cp + 源文件 + 目标路径 + 新文件名）来实现：

cp dataset.xlsx mydir/dataset-test.xlsx

五、删除文件

我们如果想要删除文件，我们需要使用的指令为rm（表示remove），比如，我们想要删除刚刚复制的dataset-test.xlsx文件，那么我们只需要指明这个文件或者这个文件的位置即可。

所以，如果我们在目录new中，我们的指令为：

rm mydir/dataset-test.xlsx

类似地，如果我们在目录mydir中，我们就不要指明路径，则我们的指令为：

rm dataset-test.xlsx

我们也同样可以用*来批量删除文件。比如，如果想要删除mydir目录下的所有文件，在目录mydir下使用的指令为：

rm *

当我们回到目录new下，并且用rm指令来删除目录（比如mydir）的时候，我们会得到一个错误「rm: mydir/: is a directory」。这表明rm指令不能被用于删除空目录。在删除空目录的时候，我们需要使用的是rmdir（表示remove directory）命令：

rmdir mydir/

但是，如果我们想要删除非空目录，那么我们就不能使用rmdir指令，比如，如果我们想要删除files目录，我们会得到一个错误「rmdir: files/: Directory not empty」。在删除非空目录及其所有文件时，我们需要使用的是加了options的rm指令。比如，如果我们要删除new目录中的files目录，我们可以输入指令：

rm -rf files/

这条指令中的选项 -r 表示逐一删除目标目录中的文件。-f 表示的是强行删除，不需要询问。因此 -rf表示的就是逐一强行删除目标目录中文件和目标目录。

六、查看手册

对于Unix-like的操作系统来说，我们可以使用一些指令来查找通用的指南。比如一个非常简短的总结性指南可以通过help命令获得。

help

此外，我们还可以通过下面的命令来获得一个更加详细的指南：

less --help

或者,

more --help

同样地，对于每一句指令，操作系统也为我们准备了详细的手册。查询他们的方式是使用man（表示manual）命令。比如，如果我们想要查询rm命令的手册，我们可以输入：

man rm

在进入所有手册后，我们都可以通过键盘的up和down键来进行翻页，我们也可以随时按字母q键（表示quit）退出手册。
