(14) Properties fo the Matrix

  • AB ≠ BA
  • A² = 0 not ⇒ A = 0, counterexample:
  • A(BC) = (AB)C
  • A(B+C) = AB + AC, while B and C should be in the same size
  • (B+C)A = BA + CA, while B and C should be in the same size
  • (r + s)A = rA + sA
