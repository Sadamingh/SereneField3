  1. A/B Testing

(1) The Definition of A/B Testing

A/B testing is a specific kind of hypothesis testing. The goal of A/B testing in business settings is to improve a certain target.

(2) Steps of A/B Testing

  • Customer Funnel
  • Define Matrix
  • Form Hypothesis
  • Formulate Test Plan
  • Run Experiments
  • Analyze Test Result
  • Conclude

2. Machine Learning Basis

(1) Unsupervised Learning vs. Supervised Learning

Supervised learning requires training labeled data. You’ll need to first label the data you’ll use to train the model to classify data into your labeled groups. Unsupervised learning, in contrast, does not require labeling data explicitly.

(2) Clustering vs. Classification

Clustering is framed in unsupervised learning, whereas classification belongs to supervised learning. So when the data is not labelled, we have to use clustering, and instead we have to use classification.

(3) KNN vs. k-means

KNN is a supervised learning with labelled data (classification), whereas k-means is an unsupervised learning without labelled data (clustering).

(4) AUC vs. ROC Curve

ROC curve is a way to measure whether a machine learning model is good or bad.

AUC is the area under the ROC curve.
